軍旅手記04

發表於 2019-06-12 更新於 2020-01-23 分類於 零碎雜文

當初從新竹關西新訓通信連受志願役調適教育結束後送到601下部隊，我印象非常深刻，當天晚上才到部一天，隔天早上就被丟到高雄步校受訓了。

最讓我寒心的是當天的編班，我硬生生的和一起被送訓過來的同梯拆散，我是161號第17班，其他六人通通都在第16班，一般通常是10個人，等於是我要和陌生的9人重新熟悉了。

步校第一天我就接實習幹部（值星）外加武管班班頭，武管班顧名思義就是負責管理武器，而班頭就是要負責武器槍枝的申請文件和親跑流程，而實習幹部的部分是每個禮拜換一個，但就是這第一個禮拜，就要把我搞死了，後來中隊上才調整為幹部人員可以不用負責勤務，也就是說如果這禮拜你背值星，如果你是武管班或器材班人員可以不用負責該班勤務。

也就是說，我第一個禮拜完全是被拿來實驗的，後面隊上觀察實際狀況做調整，我怎麼可以那麼幸運第一個禮拜就這樣苦呢？

我們那梯次有4個教授班，一二教授班一組，三四教授班一組，我在第四教授班，每個教授班都會有武管班和器材班，第一個月我們是武管班，第二個月經過調整變成器材班，此為後話。

在第一個月裡，基本上我都和第三教授班的武管班班頭一起作業，特別是剛開始的一兩個禮拜，流程、文件等等通通都不熟，在隊上辦公室弄文檔弄到超過12點甚至接近1點都是常有的事情，還弄到快2點過，好不容易弄好文檔列印出來，隔天要去軍械室跑申請流程，然後又因為出差錯被那邊的軍械士罵翻，終於申請完武器槍枝等，上完整天的課還完武器才算告一個段落，然後洗完澡休息一下下就又要開始隔天文件的處理，每次處理完文件回到寢室聽見的都是滿寢的連連鼾聲，而我卻只能在躺下時默默地留下眼淚，怨嘆著自己為何大老遠跑來受罪，這種痛苦的狀態直到第三個禮拜才漸漸好轉。

後來中隊長意識到武管班頭壓力超乎常人，強制讓班頭工作輪換下去，第二個月甚至讓武管班器材班對調，雖然器材班也是蠻累的，但對我來說已經有如天壤之別了，至少在心靈上不用承受那麼重的壓力。

步校的兩個月，整體而言對我是痛苦的，讓我撐下去的竟是對家中的思念，非常的奇妙啊哈哈。

軍旅手記03

發表於 2019-05-31 更新於 2020-01-23 分類於 零碎雜文

漢光演習終於結束了，雖然我沒有被派出去演訓，在營區內也並沒有因為漢光的關係而增加什麼勤務，反而有更輕鬆的嫌疑XD。

雖然很輕鬆，但心其實很累，尤其是和特別的人相處的時候，壓力真的很大這裡就不多談。在今天結束下來休息之後，有比平常更加鬆了一口氣的感覺。

回到寢室後洗澡，一邊聽著老歌，一邊回想著簽下志願役的事情…

我在新竹關西新訓，當時簽下去後我們幾個弟兄（簽同一個單位的）就被調去通信連做調適教育（三週），這段期間真的是非常快活，可以說基本上從新訓到通信連調適教育的這些日子，我肉體上苦的時候非常少，大概也就是這個原因，所以後面接踵而來有兩件讓我覺得非常的苦，不管是肉體還是心靈感覺都在被摧殘著。

第一件事是在通信連最後13天左右（大概）我連續燒燒退退三天,最燙有超過40度幹部才覺不妙，轉到附近的東元醫院，後來他們抽了我兩管血檢查不出原因建議我住院，但住院費太貴，所以後來轉院去淡水馬偕醫院，還讓軍中學長一路載我過去和家人會合辦理住院，真的是非常感謝。

這段期間很危險，外界和軍中都在流行A流感，但我不是，馬偕抽了我4，5管血，採集了糞便，痰，尿液等等拿去檢驗，居然都檢驗不出所以然，只有做斷層掃描得到肺炎的結論。

雖然不知道病因,但是還好醫生打的抗生素是有效的，我總共住院了7天，這段期間一樣都是燒燒退退，在最後一天終於穩定下來，決定辦理出院回營，而至今我還是不知道是什麼病毒，唯一知道的是聽主治醫師說我的血液被送疾管局了，是有沒有那麼扯…

出院那天，我和家人一起看著我的診斷報告，只見上頭寫著病因：不明、敗血症…

這七天真的很痛苦，身心俱疲，發燒咳嗽拉肚子樣樣來，爸媽輪流陪我在醫院度過這七天，期間通信連也有派人帶水果來慰問，再次對幫助我的人獻上感謝。

這是第一件事，第二件事發生在高雄，那是之後一兩個禮拜的事情了，我到部後和11位弟兄在旅部連大寢睡了一晚就馬上被丟到高雄鳳山步校受訓了，怎一個扯字了得啊 ! 這部分留待之後再說吧，等我想寫的時候再來好好琢磨一下。

做個小小結論，雖然兩件事令我身心都非常痛苦，但是不得不說彷彿一切都是上天在安排似的，藉著發燒事件之後我到了步校，另外一個階段的煎熬我幾乎是靠著家的力量讓我撐過去的，怎麼說呢？就是思念的力量吧哈哈哈。

寫到這邊，現在時間2127，漢光結束後的晚上，我在601旅部連寢室打了這篇文章，打完收工來玩手機了XD。（以後都用這種記錄時間的方式收尾吧，感覺還行喔！）

軍旅手記02

發表於 2019-05-30 更新於 2020-01-23 分類於 零碎雜文

前陣子我的識別證套壞掉了，我不得不換掉他，他的損壞程度已經達到稍微大一點的晃動我的識別證就要掉出來的程度了。

為什麼要特別提這張識別證套呢？其實也沒有什麼特別的原因，純粹只是這張識別證套伴我度過步校的2個月時間，外加下部隊到龍潭601的這段期間，算算大概4個月了，而我發現在我把證套退役下來換上新證套的時候（其實也不算新的，之後再公布新證套身份哈哈），內心真的充滿難過與惋惜，但更多的是我告訴自己一定要更努力。

半年，其實不長，但也足以改變很多事情了。我覺得自己在這段期間著實有巨大的變化，好的壞的都有，但不變的是我確確實實的朝著自己所立的方向努力著，並且也為自己可以這樣撐下去感到非常自豪。

責任是生死問題，這是我喜歡的一位老師所說的話，這句話也幾乎是我的行動準則，就算事情我努力不到10分，我也會盡量出力到6～7分，因為天底下真的沒有白吃的午餐，我一直以來都有一種冥冥之中的感覺，今天我無緣無故的受惠了，那麼在這世間的某一個角落，一定馬上有人會受到損失，我覺得這是天道的運行法則。

舉一個在軍中的常見例子，大兵手記大家應該都知道，我總是寫到八分滿，有時甚至會不夠寫。同事們都覺得我太認真，但對我來說這是屬於我的事情（事實上站在客觀的角度也確實如此），那麼做不到滿分也要做個6～7分才行，至少要覺得“嗯，這樣可以及格”才行。

這是態度，是信仰，不管喜歡不喜歡，只要事情有一兩分責任在自己身上的話，並且如果這件事你沒有出到力，如果是一個認真在過生活的人，應該要感到痛苦或悔改才對。

這件事情我喜歡，做了，叫正常。

這件事情我不喜歡，做了，叫面對。

其實是顯而易見的常理，我很慶幸，我是一個認真過生活的人，從大學時期，到現在，仍持續堅持著最初的模式。
